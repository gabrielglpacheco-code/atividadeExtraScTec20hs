Este projeto foi criado com o objetivo de garantir a aprovação para o curso de desenvolvimento de software de 300 horas do SCTEC, através de uma página de site simples, criada usando os conhecimentos adquiridos no curso rápido de introdução ao front end de 20 horas de duração.

A página foi criada para atender os requisitos de aprovação, contendo: Cabeçalho com nome e imagem de perfil do aluno, menu com links navegáveis sendo alguns deles com  redirecionamento externo, seção sobre com detalhes sobre o aluno, seção habilidades com uma tabela evidenciando as habilidades desenvolvidas durante o curso, uma pequena lista com projetos funcionais e interativos, executados durante o andamento do curso e que podem ser manuseados pelo usuário e uma seção de contato com campos para envio de mensagem e informações de contato do criador da página, além de rodapé.

As linguagens de programação utilizadas foram: HTML para construção do corpo do site, CSS para ajustes estéticos e de visualização e JavaScript para aprofundar a interação com o usuário através de funções simples.

O projeto é composto por uma pasta contendo 4 arquivos sendo eles: index.html, style.css, script.js e uma imagem de perfil do aluno e pode ser executado inicializando o arquivo index.html, incluído na pasta “Desafio Final” em um navegador de internet compatível com HTML5.
